# 📱 手機旗艦對戰 · Phone Flagship Battle

一款純前端、單一 HTML 檔的手機旗艦機「卡牌對戰」小遊戲，走明亮的寶可夢卡牌風，適合小朋友玩。挑兩支歷代旗艦，比效能、相機、螢幕、電池四項屬性，看誰是冠軍。

**▶️ 立即遊玩：<https://mansonliu.github.io/phone-battle/>**

任何網路、任何裝置（手機 / 平板 / 桌機）打開即玩，不用安裝、不用同 Wi-Fi、不用開電腦。頁面內的「📲 分享」按鈕會跳出 QR code，掃了就能把遊戲傳給朋友。

---

## 玩法

1. 從下方的**選角格子牆**點手機，加入發光的那一格（我方 🔵 / 對手 🔴），點完自動換邊。也可以按 🎲 隨機抽。
2. 按「⚔️ 開始對戰！」（或直接「🎲 隨機對戰」）。
3. 看兩支手機出招對決，分數高的獲勝。

### 對戰演出

- 開場：兩顆**火焰流星拳**從左右高速對衝、在中央爆炸相撞（格鬥遊戲風）。
- 逐項出招：效能 → 相機 → 螢幕 → 電池，每回合贏的一方射出屬性光彈打向對手，跳出傷害數字，差距大時還會「超強！」。
- 收尾：勝方蓋皇冠 👑、依領先分數給 ⭐ 星星評價，加上彩帶與勝利號角，並打出 **VICTORY!**

## 卡池（63 張，2006–2026）

- **完整年度旗艦**：Apple、Samsung、Google、Xiaomi、ASUS（含 ROG 電競系列）。
- **HTC**：2006–2018 完整旗艦史（TyTN → U12+）。
- **懷舊老機**：退出市場品牌的真實舊機，分數低、懷舊向 —— Acer、BenQ（含 BenQ-Siemens）。
- **其他代表作**：Sony、OnePlus、Huawei、vivo、OPPO 各少數經典。

> 分數為**手工相對分級，非實測跑分**：依 SoC 世代、感光元件 / 變焦、解析度 / 更新率 / 亮度、電池容量 / 快充綜合判定。每支手機的真實規格也會列在卡片上。

## 技術特色

- **單一 HTML 檔**，無任何函式庫依賴，響應式（手機 / 平板 / 桌機自動縮放），尊重 `prefers-reduced-motion`。
- **手機圖用 SVG 即時畫出**：手機背面 + 鏡頭模組，鏡頭數量還會隨相機分數變多。
- **音效全程 Web Audio 即時合成**（無音檔）：火焰撞擊、爆炸、出招、號角、印章重擊，外加約 140 BPM 的循環背景音樂。
- **CSS 3D 立體效果**：卡片隨滑鼠 / 陀螺儀立體傾斜視差與反光、選卡翻牌登場、衝撞時兩卡 3D 轉身互撞、勝者前浮敗者後傾。
