# 📱 手機旗艦對戰 · Phone Flagship Battle

一款純前端、單一 HTML 檔的手機旗艦機「鬥卡」小遊戲。挑兩支歷代旗艦，比效能、相機、螢幕、電池四項屬性的總分，看誰是真王者。

**▶️ 立即遊玩：<https://mansonliu.github.io/phone-battle/>**

任何網路、任何裝置（手機 / 平板 / 桌機）打開即玩，不用安裝、不用同 Wi-Fi、不用開電腦。頁面內的「📲 掃碼分享」按鈕會跳出 QR code，掃了就能把遊戲傳給朋友。

---

## 玩法

- **雙卡 PK**：自己各挑兩支手機，四屬性（效能 / 相機 / 螢幕 / 電池，各 0–100）比加總，滿分 400 分，高分獲勝。
- 對戰演出：雙劍飛入交擊 → 金屬碰撞＋爆炸音效＋螢幕震動 → 屬性逐項揭曉火花 → 總分跳動計數 → 勝方蓋上紅色「勝」印章 ＋ 👑 ＋ 彩帶 ＋ 號角。

## 卡池（63 張，2006–2026）

- **完整年度旗艦**：Apple、Samsung、Google、Xiaomi、ASUS（含 ROG 電競系列）。
- **HTC**：2006–2018 完整旗艦史（TyTN → U12+）。
- **懷舊老機**：退出市場品牌的真實舊機，分數低、懷舊向 —— Acer、BenQ（含 BenQ-Siemens）。
- **其他代表作**：Sony、OnePlus、Huawei、vivo、OPPO 各少數經典。

> 分數為**手工相對分級，非實測跑分**：依 SoC 世代、感光元件 / 變焦、解析度 / 更新率 / 亮度、電池容量 / 快充綜合判定。

## 技術特色

- **單一 HTML 檔**，無任何函式庫依賴，響應式 + 深色霓虹電競風。
- **音效全程 Web Audio 即時合成**（無音檔）：金屬撞擊、爆炸、號角、印章重擊，外加約 140 BPM 的循環背景音樂。
- **純 CSS 3D 立體效果**：卡片隨滑鼠 / 陀螺儀立體傾斜視差與反光、選卡翻牌登場、衝撞時兩卡 3D 轉身互撞、勝者前浮敗者後傾。
